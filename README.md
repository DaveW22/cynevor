# Cynevor Group Business Site

This repository contains the HTML5 hosted business site for **Cynevor Group**.

## Structure

```
├── index.html          # Main landing page
├── css/
│   └── styles.css      # Site stylesheet
├── js/
│   └── main.js         # Site JavaScript
└── README.md
```

## Features

- Responsive HTML5 layout with sticky navigation and mobile hamburger menu
- Hero section, About, Services, and Contact sections
- Client-side contact form validation
- Accessible markup (ARIA attributes, semantic HTML)
- Pure HTML / CSS / vanilla JS – no external dependencies required

## Getting Started

Open `index.html` in any modern web browser, or deploy the repository root to any static web host (GitHub Pages, Netlify, Vercel, etc.).
